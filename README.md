Angular Html Static
======

Midas-ng untuk membantu membuat html angular 

Syarat:

* Nodejs

Menjalankan midas-ng


Install node 
```
$ cd <midas-ng-directory>
$ npm install -g live-server
```

Run midas-ng
```
$ cd <midas-ng-directory>
$ live-server
```

Buka browser Anda, aplikasi akan tersedia di http:// 127.0.0.1:8080/
